# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-18

### Added

- Initial stable release of **JanSankalp AI**.
- Multilingual Voice-to-Text complaint registration.
- AI-powered categorization and severity assessment using GPT-4.
- Geo-spatial dashboard for administrative tracking.
- Automated department routing system.
- Real-time notifications via Pusher.
- Comprehensive documentation (/docs).
- Docker support for one-command startup.
- GitHub Actions for CI/CD, Docker verification, and Security.
