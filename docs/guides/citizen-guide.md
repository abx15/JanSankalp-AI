### What is JanSankalp AI?
JanSankalp AI is a smart civic governance platform that allows you to:
- File complaints about civic issues
- Track complaint resolution progress
- Receive real-time updates

5. **Login**: Use your email and password

## ðŸ“ Filing a Complaint

### Step-by-Step Guide

#### 1. Access Complaint Form
- **From Dashboard**: Click "File New Complaint"
- **From Menu**: Navigate to "Complaints" â†’ "File Complaint"

#### 2. Fill Complaint Details

**Basic Information:**
- **Title**: Brief description (e.g., "Pothole on Main Street")
- **Description**: Detailed explanation of the issue
- **Category**: Select appropriate category from dropdown
- **Severity**: Choose severity level (Low, Medium, High, Critical)

**Location Information:**
- **Address**: Enter the exact location
- **GPS Coordinates**: Auto-detected or enter manually
- **Landmark**: Nearby landmark for easier identification

**Photo Evidence:**
- **Upload Photo**: Take photo or upload from gallery
- **Multiple Photos**: Add up to 5 photos
- **File Types**: JPG, PNG, HEIC (max 10MB each)

#### 3. Review and Submit
- **Review**: Check all details for accuracy
- **Submit**: Click "Submit Complaint"
- **Confirmation**: Receive complaint ID (e.g., COMP-2026-0456)
