# Admin & Officer Guide

JanSankalp AI provides a data-driven dashboard for governance.

## Key Metrics

- **Total Complaints**: All reports submitted by citizens.
- **Resolved Percentage**: Ratio of resolved issues to total reports.
- **Severity Index**: AI-calculated priority (1-5).

## How to use

1. Go to **Overview** for city-wide stats.
2. Use **Complaints** to filter by department and severity.
3. Access **Map View** for geo-spatial analysis.
